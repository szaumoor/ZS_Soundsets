# ZS_Soundsets

## Introduction

This mods adds female soundsets for BG, BG2, BGEE, BG2EE, BGT, and EET for player use.

## Components

### Pious Knight (0)

This is a soundset for the "pious knight", which is a soundset for a paladin-esque class.

### Deadly rogue (1)

This is a soundset for the "deadly rogue", which is a soundset for a ruthless rogue type (assassin, shadowdancer...), though not outright evil.
